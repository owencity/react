📘 HTML 태그 + 속성 총정리

자주 쓰이는 HTML 태그들과 주요 속성을 표로 정리했습니다.

문서 구조 / 시맨틱 태그

문서 구조 태그
태그의미주요 속성특징
<!DOCTYPE html>HTML5 문서 선언-반드시 문서 첫 줄에 위치
<html>루트 요소lang="ko"문서 전체를 감쌈
<head>메타데이터 영역-meta, title, link 등 포함
<body>본문-화면에 보이는 콘텐츠
<header>머리글-사이트 상단, 섹션 시작 부분
<nav>내비게이션-메뉴/링크 모음
<main>메인 콘텐츠-한 문서에 하나만 권장
<section>구역-h2~h6 제목과 함께 사용 권장
<article>독립 글-뉴스, 블로그 글 단위
<aside>보조 정보-사이드바, 광고 등
<footer>바닥글-저작권, 연락처

<del>삭제-취소선
<ins>추가-밑줄
<sub>아래첨자-H2O
<sup>위첨자-X2

목록 태그

목록 태그
태그의미주요 속성특징
<ul>순서 없는 목록-● 불릿 기본
<ol>순서 있는 목록type="1|A|a|I|i"번호 스타일 지정
<li>목록 항목-ul/ol 자식으로 사용
<dl>정의 목록-용어-설명 세트
<dt>정의 제목-용어
<dd>정의 설명-설명

표 관련 태그
태그의미주요 속성특징
<table>표border, width표 전체
<caption>표 제목-표 설명
<thead>머리글-열 제목 행

<td>데이터 셀colspan, rowspan기본 데이터

미디어 태그

미디어 태그
태그의미주요 속성특징
<img>이미지src, alt, width, heightalt는 필수
<audio>오디오controls, autoplay, loop지원 브라우저 제한 있음
<video>비디오controls, autoplay, muted, loop포맷 호환성 주의
<source>미디어 소스src, typeaudio/video 내부에서 사용
<figure>미디어 묶음-이미지+캡션
<figcaption>캡션-figure 설명

폼 태그

폼 태그
태그의미주요 속성특징
<form>입력 폼action, method서버 전송 정보
<input>입력 필드type, name, value, placeholder, required다양한 타입 지원
<textarea>여러 줄 입력name, rows, cols, placeholder멀티라인 입력
<select>드롭다운name, multipleoption 포함
<option>선택 항목value, selectedselect 안에서 사용
<button>버튼type=submit|reset|button클릭 가능 버튼
<label>입력 레이블forinput과 연결
<fieldset>폼 그룹-여러 입력을 묶음
<legend>폼 그룹 제목-fieldset 설명

기타 태그

기타 태그
태그의미주요 속성특징
<a>하이퍼링크href, target, title페이지 이동
<address>연락처-footer에서 자주 사용
<code>코드 표시-monospace 폰트
<pre>서식 유지-공백/줄바꿈 그대로 출력
<span>인라인 영역-작은 텍스트 스타일링
<div>블록 영역-의미 없는 컨테이너
<script>자바스크립트src, type외부/내부 스크립트
<style>내부 CSS-head 안 배치
<link>외부 CSS 연결rel, hrefhead 안에서 사용
<meta>메타데이터charset, name, content, viewport인코딩/SEO/반응형
